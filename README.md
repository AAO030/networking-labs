# Networking Labs

A collection of hands-on networking labs exploring core internet protocols through direct packet capture and analysis — built while studying computer networks.

Each lab includes methodology, annotated packet-capture screenshots, and findings — the reasoning behind the answers, not just the answers themselves.

## Labs

| Lab | Topics | Key Skills |
|---|---|---|
| [Wireshark HTTP Analysis](./wireshark-http) | HTTP/1.1, caching, TCP segmentation, authentication | Packet inspection, protocol debugging, traffic analysis |
| [Password Cracking with John the Ripper](./john-the-ripper-lab) | Password hashing, dictionary/hybrid/brute-force attacks, entropy | Offensive security tooling, quantitative risk analysis |

*(More labs added as completed — TCP, DNS, TLS, and routing protocol analysis planned.)*

## About This Repo

Every lab follows the same structure:
- **Skills Demonstrated** — what this lab proves I can do
- **Methodology** — how the capture was set up
- **Findings** — evidence-backed answers, with screenshots
- **Key Takeaways** — what the results actually mean

## Tools Used Across Labs
`Wireshark` · `TCP/IP` · `Google Chrome` · packet-level protocol analysis
